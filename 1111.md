Thought for 3m 31s
Do not move away from DirectX just to solve this. The better path is to keep the renderer backend abstraction you already have and make the viewport host a real editor preview runtime, not a fake panel drawing path.

Your current repo already points that way:

WorkspaceViewportManager is designed as the viewport orchestrator. It requests slots, binds surfaces, registers scene providers, and runs a per-frame render loop.

SceneEditorTool is already set up to attach a viewport manager and delegate scene content through an IViewportSceneProvider.

NovaForgePreviewRuntime already exists specifically to provide lightweight runtime scene data to the editor viewport without a full game boot.

PCGPreviewService already exists to generate preview content into a preview world.

That means your missing piece is not “switch graphics API.” It is authority wiring.

Best architecture
Use this model:

Atlas Workspace shell
owns the window, docking, tool routing, and viewport slots.

Viewport host
owns the surface, frame loop, camera state handoff, and overlay/gizmo composition.

NovaForge preview runtime
owns preview-world state for the active editor context.

Editor panels
edit documents and push changes into the preview runtime or preview world.

Renderer backend
stays abstract. DirectX, OpenGL, or Vulkan should be an implementation detail.

Direct answer to your three options
1. Move away from DirectX to OpenGL?
No. Not as the main fix.

Why:

That is a renderer rewrite cost.

It does not solve panel-to-document wiring.

It does not solve preview-runtime ownership.

Your repo already has backend abstraction and even multiple backend files present. The bottleneck is the editor/runtime bridge, not API choice.

Only switch APIs if DirectX is proven broken in a way the abstraction cannot tolerate. Right now there is no sign that OpenGL would solve the real issue.

2. Capture runtime directly in viewport?
Yes, but not by embedding the full shipping client blindly.

Do editor preview runtime capture, not full game capture.

Meaning:

create a lightweight NovaForgePreviewRuntime

feed it the currently loaded document state

render that preview world into the viewport surface

let tools modify the document

let the document sync the preview runtime

optionally let some panels run simulation steps for live preview

That is the correct middle ground.

3. Make the viewport visually editable from panels?
Yes. That should be the core model.

Panels should not be isolated forms. They should be editors for the same live preview state. That is also consistent with your earlier pixel/tile tool direction where the center viewport is the main canvas and side panels are inspectors and controls.

bool SceneEditorTool::ensureViewportSlot() {
    if (!m_viewportMgr) return false;
    if (m_viewportHandle != kInvalidViewportHandle) return true;

    m_viewportHandle = m_viewportMgr->requestViewport(kToolId, {0.f, 0.f, 1280.f, 720.f});
    if (m_viewportHandle == kInvalidViewportHandle) return false;

    m_viewportMgr->activateViewport(m_viewportHandle);

    if (m_viewportPanel) {
        WorkspaceViewportBridge::connect(m_viewportPanel, *m_viewportMgr, m_viewportHandle);
    }

    return true;
}

void SceneEditorTool::releaseViewportSlot() {
    if (!m_viewportMgr) return;

    if (m_providerRegistered) {
        m_viewportMgr->unregisterSceneProvider(kToolId);
        m_providerRegistered = false;
    }

    if (m_viewportPanel && m_viewportHandle != kInvalidViewportHandle) {
        WorkspaceViewportBridge::disconnect(m_viewportPanel, *m_viewportMgr, m_viewportHandle);
    }

    if (m_viewportHandle != kInvalidViewportHandle) {
        m_viewportMgr->releaseViewport(m_viewportHandle);
        m_viewportHandle = kInvalidViewportHandle;
    }
}

void SceneEditorTool::syncProviderRegistration() {
    if (!m_viewportMgr || m_viewportHandle == kInvalidViewportHandle) return;

    IViewportSceneProvider* provider = m_sceneProvider ? m_sceneProvider : this;
    if (!provider) return;

    m_viewportMgr->unregisterSceneProvider(kToolId);
    m_providerRegistered = m_viewportMgr->registerSceneProvider(kToolId, provider);
}

void SceneEditorTool::setViewportPanel(UI::AtlasUI::ViewportPanel* panel) {
    m_viewportPanel = panel;
    if (m_viewportMgr && m_viewportHandle != kInvalidViewportHandle && m_viewportPanel) {
        WorkspaceViewportBridge::connect(m_viewportPanel, *m_viewportMgr, m_viewportHandle);
    }
}

    std::vector<DocumentPanelValidationMessage> messages;
};

class DocumentSavePipeline {
public:
    using NotificationCallback = std::function<void(const std::string&, const std::string&)>;

    explicit DocumentSavePipeline(NotificationCallback notify = {})
        : m_notify(std::move(notify)) {}

    void setDocumentRegistry(const NovaForgeDocumentRegistry* registry) {
        m_registry = registry;
    }

    SaveResult save(IDocumentPanel& panel) {
        SaveResult result;

        if (!panel.hasDocument()) {
            result.status = SaveResultStatus::NoDocument;
            notify("error", "[DocumentSavePipeline] No document bound to panel '" +
                            panel.panelId() + "'");
            return result;
        }

        result.messages = panel.validate();
        for (const auto& m : result.messages) {
            if (m.severity == DocumentPanelValidationSeverity::Error) {
                result.status = SaveResultStatus::ValidationFailed;
                notify("error", "[Validation] " + m.field + ": " + m.message);
            } else if (m.severity == DocumentPanelValidationSeverity::Warning) {
                notify("warning", "[Validation] " + m.field + ": " + m.message);
            }
        }
        if (result.status == SaveResultStatus::ValidationFailed) {
            return result;
        }

        NovaForgeDocument* doc = panel.boundDocument();
        if (!writeDocument(*doc)) {
            result.status = SaveResultStatus::WriteError;
            notify("error", "[DocumentSavePipeline] Write failed for '" +
                            doc->filePath() + "'");
            return result;
        }

        doc->clearDirty();

        result.status = SaveResultStatus::Ok;
        notify("info", "[DocumentSavePipeline] Saved '" + doc->filePath() + "'");
        return result;
    }

    bool revert(IDocumentPanel& panel) {
        if (!panel.hasDocument()) return false;
        bool ok = panel.revert();
        if (ok) {
            notify("info", "[DocumentSavePipeline] Reverted '" +
                           panel.boundDocument()->filePath() + "'");
        } else {
            notify("error", "[DocumentSavePipeline] Revert failed for panel '" +
                            panel.panelId() + "'");
        }
        return ok;
    }

    std::vector<DocumentPanelValidationMessage> validateOnly(const IDocumentPanel& panel) {
        auto msgs = panel.validate();
        for (const auto& m : msgs) {
            if (m.severity == DocumentPanelValidationSeverity::Error) {
                notify("error", "[Validation] " + m.field + ": " + m.message);
            } else if (m.severity == DocumentPanelValidationSeverity::Warning) {
                notify("warning", "[Validation] " + m.field + ": " + m.message);
            }
        }
        return msgs;
    }

private:
    bool writeDocument(const NovaForgeDocument& doc) {
        namespace fs = std::filesystem;
        try {
            fs::path filePath(doc.filePath());
            if (filePath.empty()) return false;

            auto parent = filePath.parent_path();
            if (!parent.empty() && !fs::exists(parent)) {
                fs::create_directories(parent);
            }

            std::ofstream out(filePath);
            if (!out.is_open()) return false;

            out << "{\n";
            out << "  \"schema\": \"novaforge.document.v2\",\n";
            out << "  \"type\": \"" << documentTypeName(doc.type()) << "\",\n";
            out << "  \"displayName\": \"" << doc.displayName() << "\",\n";
            out << "  \"dirty\": false\n";
            out << "}\n";

            return out.good();
        } catch (...) {
            return false;
        }
    }

        if (m_notify) m_notify(severity, message);

namespace NovaForge {

enum class SaveResultStatus : uint8_t {
    Ok,
    NoDocument,
    ValidationFailed,
    WriteError,
};

struct SaveResult {
    SaveResultStatus status = SaveResultStatus::Ok;
    std::vector<DocumentPanelValidationMessage> messages;
};

class DocumentSavePipeline {
public:
    using NotificationCallback = std::function<void(const std::string&, const std::string&)>;

    DocumentSavePipeline() = default;

    void setNotificationCallback(NotificationCallback cb) {
        m_notify = std::move(cb);
    }

    SaveResult save(IDocumentPanel& panel) {
        SaveResult result;

        if (!panel.hasDocument()) {
            result.status = SaveResultStatus::NoDocument;
            notify("error", "[DocumentSavePipeline] No document bound to panel '" +
                            panel.panelId() + "'");
            return result;
        }

        result.messages = panel.validate();
        for (const auto& m : result.messages) {
            if (m.severity == DocumentPanelValidationSeverity::Error) {
                result.status = SaveResultStatus::ValidationFailed;
                notify("error", "[Validation] " + m.field + ": " + m.message);
            } else if (m.severity == DocumentPanelValidationSeverity::Warning) {
                notify("warning", "[Validation] " + m.field + ": " + m.message);
            }
        }
        if (result.status == SaveResultStatus::ValidationFailed) {
            return result;
        }

        NovaForgeDocument* doc = panel.boundDocument();
        if (!writeDocument(*doc)) {
            result.status = SaveResultStatus::WriteError;
            notify("error", "[DocumentSavePipeline] Write failed for '" +
                            doc->filePath() + "'");
            return result;
        }

        doc->clearDirty();

        result.status = SaveResultStatus::Ok;
        notify("info", "[DocumentSavePipeline] Saved '" + doc->filePath() + "'");
        return result;
    }

    bool revert(IDocumentPanel& panel) {
        if (!panel.hasDocument()) return false;
        bool ok = panel.revert();
        if (ok) {
            notify("info", "[DocumentSavePipeline] Reverted '" +
                           panel.boundDocument()->filePath() + "'");
        } else {
            notify("error", "[DocumentSavePipeline] Revert failed for panel '" +
                            panel.panelId() + "'");
        }
        return ok;
    }

    std::vector<DocumentPanelValidationMessage> validateOnly(const IDocumentPanel& panel) {
        auto msgs = panel.validate();
        for (const auto& m : msgs) {
            if (m.severity == DocumentPanelValidationSeverity::Error) {
                notify("error", "[Validation] " + m.field + ": " + m.message);
            } else if (m.severity == DocumentPanelValidationSeverity::Warning) {
                notify("warning", "[Validation] " + m.field + ": " + m.message);
            }
        }
        return msgs;
    }

private:
    bool writeDocument(const NovaForgeDocument& doc) {
        namespace fs = std::filesystem;
        try {
            fs::path filePath(doc.filePath());
            if (filePath.empty()) return false;

            auto parent = filePath.parent_path();
            if (!parent.empty() && !fs::exists(parent)) {
                fs::create_directories(parent);
            }

            std::ofstream out(filePath);
            if (!out.is_open()) return false;

            out << "{\n";
            out << "  \"schema\": \"novaforge.document.v2\",\n";
            out << "  \"type\": \"" << documentTypeName(doc.type()) << "\",\n";
            out << "  \"displayName\": \"" << doc.displayName() << "\",\n";
            out << "  \"dirty\": false\n";
            out << "}\n";

            return out.good();
        } catch (...) {
            return false;
