# Repo Status Legend

Use these fixed labels only. Do not use vague alternatives.

## Status Labels

| Label | Meaning |
|-------|---------|
| **Done** | Fully implemented, tested, documented |
| **In Progress** | Actively being worked on |
| **Partial** | Some functionality exists but incomplete |
| **Planned** | Described but no implementation exists |
| **Blocked** | Cannot proceed due to dependency |
| **Archived** | Retired from active development |

## Do Not Use

- "basically done"
- "mostly there"
- "nearly complete"
- "kind of working"
- "should work"
- "needs minor fixes"

These are audit poison. Use the strict labels above.
